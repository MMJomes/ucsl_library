## Installation

1. Create .env file under the project root folder and copy paste code from .env.example

2. Run composer install command
```php
composer install 
```

2. Run npm install
```php
npm install
```

3. Run npm run dev
```php
npm run dev 
```

4. ready to use
```php
php artisan serve
```
